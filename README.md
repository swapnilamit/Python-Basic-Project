# Python Terminal To-Do List

A simple, interactive command-line interface (CLI) to-do list application written in Python. This project is a great basic project for learning Python fundamentals, including user input, loops, lists, and basic control flow.

## Features

- **Add Tasks**: Quickly append new tasks to your list.
- **Show Tasks**: View your current list of tasks.
- **Update Tasks**: Edit existing tasks by their list number.
- **Delete Tasks**: Remove tasks from the list by their task number.
- **Input Validation**: Safely handles invalid choices.

## Prerequisites

To run this application, you only need Python installed on your system:
- Python 3.x

## How to Run

1. Clone or download the repository to your local machine.
2. Open your terminal/command prompt and navigate to the project directory.
3. Run the script:
   ```bash
   python to_do_list_basic.py
   ```

## Usage

When you run the script, you'll be greeted with the main menu:

```text
WELCOME TO THE TO DO LIST
.........................
Option 1: Add Task
Option 2: Show Task
Option 3: Update Task
Option 4: Delete Task
Option 5: Close To Do List
```

Simply enter the option number (`1` to `5`) to perform the corresponding action.
